# 이미지 업로드 가이드

이 폴더에 웹사이트에서 사용할 이미지 파일을 업로드하세요.

## 프로필 이미지 (profile)

### 지원 형식
- profile.jpg
- profile.png
- profile.jpeg
- profile.webp

### 사용 방법
1. 위 형식 중 하나로 이미지 파일을 준비하세요
2. 파일명을 `profile.jpg` (또는 다른 확장자)로 저장하세요
3. 이 폴더에 업로드하세요
4. 페이지를 새로고침하면 이미지가 표시됩니다

### 이미지 권장 사항
- 정사각형 비율 (1:1)
- 최소 크기: 400x400px
- 최적 크기: 800x800px 이상
- 파일 크기: 2MB 이하 권장

## 로고 이미지 (logo)

### 지원 형식
- logo.png
- logo.jpg
- logo.jpeg
- logo.webp
- logo.svg

### 사용 방법
1. 위 형식 중 하나로 로고 이미지 파일을 준비하세요
2. 파일명을 `logo.png` (또는 다른 확장자)로 저장하세요
3. 이 폴더에 업로드하세요
4. 페이지를 새로고침하면 헤더 좌측 상단에 로고가 표시됩니다
5. 이미지가 없으면 텍스트 로고가 표시됩니다

### 로고 권장 사항
- 가로형 비율 권장 (예: 200x60px, 300x90px)
- 배경 투명 PNG 또는 SVG 권장
- 최소 크기: 150x45px
- 최적 크기: 200x60px ~ 300x90px
- 파일 크기: 500KB 이하 권장
